# Ahmad's Football Profile

A clean, responsive profile page built using HTML5 and Tailwind CSS. 

## Features
* Centered text layout with a soft blue background.
* Modern typography styling using utility-first classes.
* Lightweight setup running entirely via CDN.

Open `index.html` in any browser to view the live webpage.
